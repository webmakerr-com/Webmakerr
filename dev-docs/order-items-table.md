## Order Items Table


**quantity**
quantity ordered for this item

**unit_price**
Actual price of the item

**cost**
If there is manage cost then cost of the item else 0

**subtotal**
quantity * unit_price

**tax_amount**
Total tax applied on this item

**discount_total**
Total discounted coupon amount for this item

**line_total**
subtotal - discount_total + tax_amount

**refund_total**
Total amount refunded for this item
